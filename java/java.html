    • js写成一行的原因？
             js文件的空格占字节，写成一行能缩小文件大小. 文件越大效果越好.用Eclipse,可以下载插件Aptana,安装完后工具栏多了一个选项:Scripts ---> Javascript ----> Compact
    • 如何不占空间地隐藏与显示div？
             设置div的style="display: none;"，不能在css里面写，否则 会永久隐藏。然后在js中创建鼠标事件
      $('.general').mouseover(function(){
      $(this).find(' > div')[0].style.display="";
      })
      $('.general').mouseout(function(){
      $(this).find(' > div')[0].style.display="none";
      })
    • 为何有时候往节点中append字符串时会只放字符串？
              节点如果是从class中获取会追加字符串，所以要从id中获取。
    • css常用设计
      a标签不要下划线：text-decoration: none;
      设置相对定位：position: relative;
      设置外边距：margin: auto auto;
      设置内边距：padding:10px 20px;
      溢出部分处理：overflow: hidden(修剪)/auto(滚动条)/visible(显示在框外)/scroll/inherit/visible;
      内容尺寸：zoom:normal;(使用实际尺寸)
      设置光标（如a标签）：cursor:auto;
      设置列表属性：list-style:none;list-style-type:none;
      设置文本格式：text-align:justify;(两端对齐)
      设置文本大小写：text-transform:none/capitalize(每个单词以大写字母开头)/uppercase(全部大写)/lowercase(全部小写)/inherit(继承 text-transform 属性)
      元素的堆叠顺序:z-index:0;(Z-index 可为负值，仅能在定位元素上奏效)
      背景图片显示形式：background-repeat:no-repeat;
      清除浮动元素：clear:both;
    • js常用设计
      获取元素id值：object.attr('id');
      改变元素状态：object.toggle();
      获取元素的父：object.parent('ul');
      获取元素的子：object.find('>li');
      判断是否包含：object.is('has(li)');
      添加元素属性：object.attr('id','123');有的元素使用：object.id;
      json对象转化：jsonArray = eval('('+jsonString+')');jsonString = JSON.stringify(jsonArray);
      强制性不换行：height:22px; overflow:hidden;加入高度和设置隐藏超出宽度文字;
      鼠标移入移出：object.onmouseover = function(){};object.onmouseout = function(){};
      visibility ：object.style.visibility = 'hidden';object.style.visibility = 'visible';
      display ：object.style.display = 'none';object.style.display = '';
  • 名词
    • 灰度发布（又名金丝雀发布）？
             指在黑与白之间，能够平滑过渡的一种发布方式。在其上可以进行A/B testing， 即让一部分用户继续用产品特性A，一部分用户开始用产品特性B，如果用户对B 没有什么反对意见，那么逐步扩大范围，把所有用户都迁移到B上面来。灰度发 布可以保证整体系统的稳定，在初始灰度的时候就可以发现、调整问题，以保证其影响度。
